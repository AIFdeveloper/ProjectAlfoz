
Для чего предназначена программа Microsoft Word?
A: Для работы с таблицами
B: Для создания и редактирования текстовых документов
C: Для просмотра видео
D: Для работы с базами данных
ANSWER: B

Какое расширение имеет файл Word по умолчанию?
A: .txt
B: .docx
C: .xlsx
D: .pptx
ANSWER: B

Какая комбинация клавиш сохраняет документ?
A: Ctrl + C
B: Ctrl + V
C: Ctrl + S
D: Ctrl + Z
ANSWER: C

Какая кнопка используется для изменения шрифта?
A: Вставка
B: Главная
C: Рецензирование
D: Вид
ANSWER: B

Что делает команда «Вырезать»?
A: Копирует текст
B: Удаляет документ
C: Перемещает текст в буфер обмена
D: Печатает текст
ANSWER: C

Какая комбинация клавиш копирует текст?
A: Ctrl + X
B: Ctrl + V
C: Ctrl + C
D: Ctrl + A
ANSWER: C

B: Для изменения цвета

D: Для выравнивания

Как выровнять текст по центру?
A: Ctrl + L

C: Ctrl + E
D: Ctrl + J

Где можно изменить межстрочный интервал?
